How to run the demo on Wii

Copy the folder into the apps/ folder on the SD card of your Wii,
so that the file structure is

SD Card root / 
- apps / 
--  keir_marmot_v11/
---  boot.dol
---  icon.png
---  meta.xml
---  assets/
----  all the asset files

How to run the demo in Dolphin

Select Option/Configuration/Wii and make the 
SD Sync Folder to point in the keir_marmot_v11 folder.
Run the boot.dol
